Tips and tricks

To make life easier for anyone seriously using Salt as a terminal, I've collected some ideas here.

Terminfo

Many terminals have been produced, and even more emulators exist for these 'glass' terminals. The capabilities of each of these are different of course. In the old days, software used to consult the termcap database to find out what the current terminal was capable of. The current terminal, that's what is defined in the TERM environment variable.

The old termcap file has been replaced on modern Unix computer with terminfo. There's a good manual page for terminfo (5) and if you download the source code you can have a look at a very large terminfo source file containing descriptions for many terminals. The usual Linux distribution these days comes with a subset of this large set of terminal definitions.

Salt is a vt220 emulator. That means it is somewhere in between the most dumb and most fancy terminal types. But Salt has mapped two keys that are not defined in the vt220 terminfo entry, the Home and End keys. Not all software uses terminfo strictly and e.g. Bash recognizes the Home and End keys sent by Salt without a problem. But e.g. Vim does not, and so we have to hack the terminfo databank a little. Fortunately, that's very easy.

What we need is a very small terminfo source file:

# This is a terminfo source file for the Salt-2.0 Terminal Emulator # salt|Salt Terminal Emulator, khome=\E[1~,kend=\E[4~, use=vt220,

Copy that source and paste it in a file, e.g. salt.src. Then execute the command:

tic salt.src

Assuming you're not root, this will create a ./terminfo directory in your HOME. In this directory you can find the compiled salt terminfo entry. To use this terminfo entry, you can change the TERM environmental variable to 'salt', e.g. by editing your .profile or .bashrc. Alternatively you can change the TERM string in the Preferences dialog of Salt, but I don't recommend that.

Salt and Vim

If you've read and followed the Terminfo tip above, you have a terminfo entry for salt by now. If you don't change the TERM variable, you still can use the terminfo entry by setting term in vim:

:set term=salt

You can do this from your .vimrc file as well.
